## Setup Instructions

1. **Clone the Repository**

```bash
git clone [repository_url](https://github.com/JoshieBoy454/CapeTownMunicipalityApp.git)
```

2. **Open Project in Android Studio**

3. **Build the Project**

   * Visual Studio will automatically download dependencies.

4. **Run the App**
   * Click **Run**.
